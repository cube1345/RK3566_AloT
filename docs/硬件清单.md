# Agent ZX 硬件清单 (Pi 5 默认配置)

> 传感器方案: SGP30 (CO₂) + DHT11 (温湿度) + BH1750 (光照) + HC-SR501 (人体)
> 目标平台: Raspberry Pi 5 8GB
> 预估总价: ¥600-750

---

## 一、核心平台

| 序号 | 名称 | 规格 | 数量 | 单价(约¥) | 用途 |
|------|------|------|------|-----------|------|
| 1 | Raspberry Pi 5 | 8GB RAM, BCM2712 | 1 | 450-500 | 主控制器 |
| 2 | MicroSD 卡 | 64GB+, A2级 | 1 | 40-60 | 系统+模型(941MB)+数据 |
| 3 | 电源适配器 | 5V/5A, USB-C, 官方推荐 | 1 | 60-80 | Pi 5 供电 |
| 4 | 散热片/风扇 | Pi 5 专用 | 1 | 30-50 | 散热(LLM推理CPU满载) |

## 二、传感器

| 序号 | 名称 | 型号 | 协议 | 地址/引脚 | 数量 | 单价(约¥) | 用途 |
|------|------|------|------|-----------|------|-----------|------|
| 5 | CO₂+TVOC 传感器 | SGP30 模块 | I2C | 0x58 | 1 | 25-35 | 二氧化碳浓度 |
| 6 | 温湿度传感器 | DHT11 模块 | GPIO | BCM4/Pin7 | 1 | 3-5 | 温度+湿度 |
| 7 | 光照传感器 | BH1750 模块 | I2C | 0x23 | 1 | 5-10 | 环境光强度 |
| 8 | 人体红外传感器 | HC-SR501 | GPIO | BCM25/Pin22 | 1 | 5-8 | 人体存在检测 |

## 三、执行器 (3继电器 + 1红外)

| 序号 | 名称 | 规格 | GPIO | 数量 | 单价(约¥) | 控制设备 |
|------|------|------|------|------|-----------|------|
| 9 | 继电器模块 | 5V低电平触发, 3路 | BCM17 | 1 | 10-15 | 灯 |
| | | | BCM27 | | | 风扇 |
| | | | BCM10 | | | 空气净化器 |
| 10 | 红外发射管 | 5mm, 940nm | BCM22 | 1 | 1-2 | 空调遥控 |
| 11 | NPN 三极管 | S8050 或 2N2222 | — | 1 | 1 | 红外管驱动 |
| 12 | 电阻 100Ω | 1/4W | — | 1 | 0.1 | 红外管限流 |

> **说明**: 空调走红外遥控，不走继电器。继电器只控制灯(BCM17)、风扇(BCM27)、净化器(BCM10)三路。单继电器只能通断，不能调亮度/转速/档位。

## 六、接线总览

```
                       Pi 5 40-pin GPIO
                    ┌──────────────────────────┐
    SGP30 SDA ──────┤ Pin 3  (SDA1, I2C)      │
    SGP30 SCL ──────┤ Pin 5  (SCL1, I2C)      │
    BH1750 SDA ─────┤ Pin 3  (共享 I2C)        │
    BH1750 SCL ─────┤ Pin 5  (共享 I2C)        │
    DHT11 DATA ─────┤ Pin 7  (BCM4)            │
    继电器 IN1·灯 ──┤ Pin 11 (BCM17)           │
    继电器 IN2·风扇 ┤ Pin 13 (BCM27)           │
    红外发射管 ─────┤ Pin 15 (BCM22)  [三极管] │
    LED 绿灯 ───────┤ Pin 16 (BCM23)  [220Ω]   │
    LED 红灯 ───────┤ Pin 18 (BCM24)  [220Ω]   │
    继电器 IN3·净化 ┤ Pin 19 (BCM10)           │
    门铃按键 ───────┤ Pin 21 (BCM9)   [10kΩ上拉]│
    HC-SR501 OUT ───┤ Pin 22 (BCM25)           │
    ─────────────────┤                          │
    3.3V ───────────┤ Pin 1/17  → SGP30, BH1750, DHT11
    5V ─────────────┤ Pin 2/4   → HC-SR501, 继电器模块
    GND ────────────┤ Pin 6/9/14/20/25 → 所有模块共地
                    └──────────────────────────┘
```

## 八、不买硬件也能跑

设 `AGENT_MOCK=1`，全部传感器用虚拟数据，Dashboard + LLM 决策 + 场景识别全部功能在 Pi 5 上可演示，无需任何外接硬件。

只需: Pi 5 + SD卡 + 电源 + 网线 (共约 ¥550-600)
